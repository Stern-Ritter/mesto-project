# Проект: Место

### Описание проекта
Проект позволяет делиться своими фотографиями с другими пользователями:
добавлять и удалять фотографии, редактировать их описание, а также ставить
фотографиям лайки.

### Использванные технологии 
* html, 
* css, 
* flex,
* grid,
* БЭМ, 
* адаптивная вёрстка,
* модальные окна,
* JavaScript.
